# Schilling Widgets System Demo

This is a React + Vite demonstration project showcasing the **Schilling Widgets System** package - a complete UI components library for React applications with TypeScript.

## 🚀 Features Demonstrated

### 📦 Component Library

-   **Basic Components**: Button variants, Card layouts, Input fields, Badges
-   **Advanced Components**: Accordion, Tabs navigation, Theme provider
-   **Advanced Widgets**: TaskManager with virtualization and inline editing

### 🎛️ TaskManager Widget

-   **Task Management**: View, edit, and manage tasks with a modern interface
-   **Real-time Updates**: Live task updates with instant UI feedback
-   **Status Management**: Track task progress with visual status indicators
-   **Priority System**: High, medium, and low priority tasks with color coding
-   **Tag System**: Organize tasks with customizable tags
-   **Inline Editing**: Edit tasks directly in the table interface
-   **Virtualization**: Handle large datasets efficiently with react-window
-   **Responsive Design**: Optimized for both desktop and mobile devices
-   **Accessibility**: Full keyboard navigation and screen reader support

## 🛠️ Technology Stack

-   **React 19** - Latest React with modern features
-   **Vite** - Fast build tool and development server
-   **TypeScript** - Type-safe development
-   **Schilling Widgets System** - Complete UI component library

## 📦 Package Information

This demo uses the `schilling-widgets-system` package which includes:

-   Complete Shadcn UI component library
-   Advanced TaskManager and InfiniteTable widgets
-   TanStack Query integration for data management
-   Flexible theming system (light/dark mode)
-   Works with or without Tailwind CSS
-   TypeScript-first development experience
-   Tree-shaking support for optimal bundle size

## 🚀 Getting Started

### Prerequisites

-   Node.js (version 18 or higher)
-   npm or yarn package manager

### Installation

1. Clone the repository:

```bash
git clone [repository-url]
cd schilling-widgets-demo
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## 📝 Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in development mode with hot reloading using Vite.

### `npm run build`

Builds the app for production. TypeScript compilation + Vite build.

### `npm run preview`

Preview the production build locally.

### `npm run lint`

Runs ESLint to check for code quality issues.
